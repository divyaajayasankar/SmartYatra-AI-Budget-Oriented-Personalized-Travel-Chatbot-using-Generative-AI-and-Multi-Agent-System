# 🇮🇳 AI Travel Chatbot
# Project Description

## Budget Oriented Personalized Travel Chatbot using Generative AI and Multi-Agent System

The proposed project is an AI-powered travel chatbot developed to help users plan budget-friendly trips across India. The system uses Generative AI, Large Language Models (LLMs), Natural Language Processing (NLP), and Multi-Agent Systems to provide personalized travel recommendations, budget estimation, safety analysis, and itinerary generation.

The chatbot accepts user inputs such as:

* Destination
* Budget
* Number of days
* Number of travelers

Based on the user query, the system intelligently analyzes a large Indian tourism dataset and recommends suitable tourist places nearby the selected destination. It also calculates estimated expenses for:

* Hotel accommodation
* Food
* Transportation
* Miscellaneous expenses

The system provides additional features such as:

* Women safety analysis
* Nearby good and risky places
* Railway station distance
* Bus stand distance
* Airport distance
* Hotel recommendations
* Day-wise itinerary planning

The project is developed using:

* ReactJS for frontend
* FastAPI for backend
* Python for AI processing
* Pandas for dataset handling
* Leaflet/OpenStreetMap for map integration

The chatbot improves travel planning by offering an intelligent, interactive, and budget-oriented travel assistant for Indian tourists. The system is scalable and can be further enhanced using real-time APIs such as Google Maps API, IRCTC API, hotel booking APIs, and weather APIs.
